## Setup virtual enviroment
1. creating <env-name> enviroment
> bash```python3 -m venv <env-name>```
2. activating enviroment
> bash```source <env-name>/bin/activate```
3. deactivating enviroment
> bash```deactivate```

## Setup requirements:
> bash```source <env-name>/bin/activate```
> bash```pip install -r requirements.txt```
