## Setup virtual enviroment
### creating <env-name> enviroment
> python3 -m venv <env-name>
### activating enviroment
> source <env-name>/bin/activate
### deactivating enviroment
> deactivate

## Setup requirements:
> source <env-name>/bin/activate
> pip install -r requirements.txt
